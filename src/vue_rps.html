VUE: RPS

First time understanding how a frame work worked with js files

Netlify

Description

This project I did for my WEB 2630 frameworks class. It was a web application to replicate a rock paper scissors game. The user would choose between rock, paper or scissors and after making their choice the application would choose a response for the computer. It would then show who won the match and increase the points and meter for either the user or the computer. After 10 games were played it would then choose a winner based of who scored the most points.

Relevence

I chose this project to show case my knowledge of knowing multiple JavaScript libraries. I have really been excited to learn the VUE framework because of all the small similar nuances it has to the REACT framework. That and learning things that I already have somewhat knowledge in really makes me giddy. This project in particular was extra exciting because it essentially was “game-ifying” the VUE framework with a really simplistic task.
