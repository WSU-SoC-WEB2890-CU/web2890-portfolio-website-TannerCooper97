Project created for a current ongoing client to help build relationship between consumer and developer.

Description

This project I did for my current WEB 2098 class. I initially had the idea to do a redesign of my aunts photography website in my WEB 2500 UX and UI design class. So I wanted to bring it to life while in this class. It features a landing page, about page, a fully functioning gallery, service pacakges, and a way of contact.

Relevence

I chose to spotlight this project because its something I really have enjoyed doing for quite some time. It still needs some heavy work, and I have the work cut out for me but I can't wait to see the fully developed project. This has given a reason to go back into react and review the framework along with teach me how I might do more higher level front end UX and UI design. I hope to see this project one day be used in the real world seen by thousands of people, because my aunt deserves to be seen.
