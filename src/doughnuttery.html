Doughnuttery

Understanding the buildng blocks of front end UX and UI design.

Description

This project I did for my WEB 1400 class. This project was designed in order to understand the building blocks of basic web HTML and CSS website creation. The premise was to redesign a page for a website to make it look more presentable and customer intuitive. Doughnuttery was an online retailer selling shipped doughnuts to the world, and I wanted to give it a more fun and Willy Wonka look to the web application.

Relevence

I wanted to included this web project to show my progress to the public. Most people would say you should put your best foot forward on every project on a portfolio but in my own opinion I really enjoy seeing the progress I have made to get where I am. If you can’t like the worst parts of you then how will you ever like the best parts of you? It may look a bit “crunchy and corny” as I would say, but I think this was the start in major steps into actually enjoying front end UI development.
