Description

This project was made during my time at DevMountain. It was designed within REACT to help others learn the REACT library. It consisted of using a node server, a PostgreSQL database and 6 different components in order to create a single page loading web application. This project specifically I created myself with the help of a few dev mountain tutors. I personally wanted to create this website to game-ify the boring parts of learning how to code in the react library .
